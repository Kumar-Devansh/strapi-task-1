
## Local Setup Performed

- Cloned the official Strapi repository from https://github.com/strapi/strapi
- Installed dependencies using `npm install`
- Ran Strapi locally using `npm run develop`
- Accessed the Admin Panel at `http://localhost:1337/admin`
- Created a sample content type named **Joke** with fields:
  - title
  - content
  - category
- Added sample entries using Content Manager
